# AWS EC2 US-East-1 区域价格参考 (2026版)

## T3 系列实例 (通用型)
- **t3.micro**: 2 vCPU, 1 GiB 内存. 价格: $0.0104 / 小时 (约 $7.50 / 月)
- **t3.small**: 2 vCPU, 2 GiB 内存. 价格: $0.0208 / 小时 (约 $15.00 / 月)
- **t3.medium**: 2 vCPU, 4 GiB 内存. 价格: $0.0416 / 小时 (约 $30.00 / 月)
- **t3.large**: 2 vCPU, 8 GiB 内存. 价格: $0.0832 / 小时 (约 $60.00 / 月)
- **t3.xlarge**: 4 vCPU, 16 GiB 内存. 价格: $0.1664 / 小时 (约 $120.00 / 月)
- **t3.2xlarge**: 8 vCPU, 32 GiB 内存. 价格: $0.3328 / 小时 (约 $240.00 / 月)

## 降级建议标准
- 如果历史平均 CPU 利用率 < 10%，且峰值不超过 30%，建议至少降级两档。
- 如果内存使用率 < 20%，必须考虑降级。
